# Техническое Задание (ТЗ): Панель управления TauTan Secure Key (v4.0)

## 1. Общее описание
Разработка веб-интерфейса (Single Page Application) для управления аппаратным ключом безопасности **TauTan Secure Key v2**. Приложение должно обладать премиальным дизайном в стиле "Cyber-Security Dark", обеспечивать интуитивное взаимодействие с устройством и предоставлять доступ к встроенному AI-консультанту.

**Текущая версия макета:** v4.0 (Final Concept)
**Целевая платформа:** Веб-браузеры (Chrome, Firefox, Safari) с поддержкой WebHID/WebAuthn.

## 2. Дизайн-система и UI/UX
### Цветовая палитра
*   **Основной фон:** Deep Black (`#050505` или `#000000`) с использованием текстуры шума (`noise-bg`).
*   **Акцентный цвет:** Coral Orange (`#EE5D43`). Используется для CTA-кнопок, важных уведомлений, брендинга.
*   **Статусные цвета:**
    *   **Успех/Активно:** Neon Green (`text-green-500`).
    *   **Ошибка/Опасно:** Red (`text-red-500`).
*   **Второстепенные элементы:** Zinc Grays (`zinc-900`, `zinc-500`) для карточек и неактивного текста.

### Визуальные эффекты
*   **Glassmorphism:** Полупрозрачные панели (`bg-zinc-900/40`, `backdrop-filter: blur`).
*   **Animations:** Плавные переходы страниц (`animate-in fade-in`), пульсация активных элементов, бегущая строка (marquee).
*   **Шрифты:** Sans-serif (Inter или системные шрифты), моноширинные шрифты для логов и технических данных.

## 3. Структура интерфейса

### 3.1. Sidebar (Боковая панель)
*   **Логотип:** Иконка Щита (Shield) в оранжевом квадрате + текст "TAUTAN".
*   **Меню навигации (7 пунктов):**
    1.  Дашборд (`LayoutDashboard`)
    2.  Управление PIN (`Key`)
    3.  FIDO2 Аккаунты (`Shield`)
    4.  Информация (`Info`)
    5.  Советы безопасности (`BookOpen`)
    6.  Новости киберрисков (`AlertCircle`)
    7.  Поддержка (`HelpCircle`)
*   **Футер меню:** Кнопка "Выход" (`LogOut`).
*   **Поведение:** Адаптивное. На десктопе сворачивается/разворачивается, на мобильных скрыта под гамбургер-меню.

### 3.2. Header (Шапка)
*   **Центр (Desktop):** Индикатор статуса "СТАТУС TAUTAN SECURE KEY: ПОДКЛЮЧЕН" (зеленый текст).
*   **Мобильная версия:** Логотип и кнопка меню.

### 3.3. Footer (Подвал области контента)
*   **Бегущая строка:** Оранжевая полоса с повторяющимся текстом: "PROTECTED BY TSARKA AI SECURITY SOLUTIONS" с иконкой щита.

### 3.4. AI Panel (Выдвижная панель)
*   **Расположение:** Справа, выдвигается поверх контента (`z-50`).
*   **Функционал:** Чат с AI-экспертом.
*   **Состояния:**
    *   Ожидание: Приветственное сообщение.
    *   Загрузка: Спиннер и пульсирующий текст "Анализ безопасности...".
    *   Ответ: Текстовый блок с ответом от API.

## 4. Функциональные требования к страницам

### А. Дашборд (Главная)
*   **Главная карточка:**
    *   Заголовок "TauTan Key v2".
    *   Статус "Устройство активно" (с зеленой точкой).
    *   Кнопка "Проверить обновления" (Primary, Coral).
    *   Кнопка "AI Анализ" (Secondary, Ghost). При клике открывает правую AI-панель с запросом про безопасность.
*   **Виджеты статистики:**
    *   "Записей FIDO2": Число (например, 14).
    *   "Уровень доверия": Текст "MAXIMUM" (оранжевый).
*   **Карточки возможностей (Grid 3 col):**
    1.  Шифрование данных (иконка Lock).
    2.  WebAuthn/FIDO2 (иконка Fingerprint).
    3.  Защита от фишинга (иконка Shield).

### Б. Управление PIN
*   **Макет:** Grid 2 колонки.
*   **Блок "Смена PIN-кода":**
    *   Поля ввода: "Текущий PIN", "Новый PIN".
    *   Кнопка "Обновить".
*   **Блок "Сброс до заводских":**
    *   Стиль: Бордер с красным оттенком.
    *   Предупреждение: "ВНИМАНИЕ: Все учетные записи FIDO2 будут безвозвратно удалены."
    *   Кнопка "Reset Device" (красная обводка/текст).

### В. FIDO2 Аккаунты
*   **Header:** Заголовок и кнопка "Регистрация" (открывает мастер настройки).
*   **Список аккаунтов:**
    *   Карточки с логотипом сервиса (Generic Shield или лого), названием и ID (domen.com).
    *   Кнопка удалелния (Крестик) для каждой записи.
*   **Мастер регистрации (Wizard):**
    *   Степ 1: Кнопка "Регистрация".
    *   Степ 2: Анимация просьбы коснуться ключа/сенсора.

### Г. Информация
*   **Блок "Системные логи":**
    *   Стилизация под терминал (моноширинный шрифт, мелкий кегль).
    *   Пример логов: Secure Boot initialized, Root of Trust established.
*   **Спецификация:**
    *   Модель: TT-SEC-X PRO.
    *   Серийный номер: EDS88000FKZ78 (выделен зеленым).
*   **Действия:** Кнопка "Проверить обновление".

### Д. Поддержка / FAQ
*   CTA-блок: "Нужна помощь?" с кнопкой "Спросить ИИ", которая открывает AI-панель с контекстом FAQ.

## 5. Технический стек (Рекомендованный)
*   **Frontend:** React 18+, TypeScript.
*   **Styling:** Tailwind CSS (обязательно использование утилит для анимаций и цветов).
*   **Icons:** Lucide React.
*   **Сборка:** Vite.
*   **Интеграция AI:** Google Gemini API (через Proxy-сервер для безопасности ключей).
